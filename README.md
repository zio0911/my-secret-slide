# My Secret Slide Viewer

개인용 이미지 뷰어 + 시크릿 스토리지 기능을 포함한
macOS Electron 기반 로컬 이미지 뷰어 애플리케이션.

유료 이미지 뷰어의 과도한 기능과 비용을 피하고,
**"나만 아는 시크릿 이미지 관리"**를 목적으로 제작.

---

## 1. 프로젝트 목표

* 로컬 폴더 기반 이미지 뷰어
* 키보드 중심 조작
* 시크릿 이미지 암호화 저장
* 가짜 비밀번호(Decoy) 확장 가능 구조
* 네트워크 의존 없음
* 개인 단독 사용 전제

---

## 2. 기본 기능

### 📁 폴더

* 로컬 폴더 선택
* 지원 이미지 확장자

    * jpg, jpeg, png, gif, webp

### 🖼 이미지 뷰어

* 좌 / 우 방향키: 이전 · 다음 이미지
* Space: 슬라이드쇼 재생 / 정지
* 자동 슬라이드 간격: 2초

---

## 3. 확대 / 이동 기능

### 🔍 확대 / 축소

* `=` : 확대
* `-` : 축소
* 최소 배율: 1
* 최대 배율: 5

### 🧭 이동 규칙

* **확대 상태(scale > 1)**

    * 방향키 → 이미지 내부 이동
* **확대되지 않은 상태(scale = 1)**

    * 좌 / 우 → 이미지 전환

확대 배율이 1로 돌아오면 위치 자동 초기화.

---

## 4. 시크릿 모드 개념

### 🔐 시크릿 파일

* 확장자: `.secret`
* 원본 이미지를 암호화한 파일
* 일반 모드에서는 노출되지 않음

### 📂 모드 구분

| 모드     | 표시 이미지       |
| ------ | ------------ |
| 일반 모드  | 일반 이미지 파일    |
| 시크릿 모드 | `.secret` 파일 |

---

## 5. 키패드 & 비밀번호 흐름

### 키패드 호출 목적

* `ENCRYPT` : 현재 이미지 암호화
* `ENTER_SECRET` : 시크릿 모드 진입

### 비밀번호 입력 후 동작

| 목적           | 결과                       |
| ------------ | ------------------------ |
| ENCRYPT      | 현재 이미지를 `.secret` 파일로 변환 |
| ENTER_SECRET | 시크릿 모드 진입                |

**중요 사항**

* `H` 키: 암호화 전용 (모드 전환 없음)
* `S` 키: 시크릿 모드 진입 전용

---

## 6. 단축키 정리

### 📌 일반 모드

| 키     | 동작            |
| ----- | ------------- |
| ← / → | 이미지 이동        |
| Space | 슬라이드쇼 시작 / 정지 |
| H     | 현재 이미지 암호화    |
| S     | 시크릿 모드 진입     |
| = / - | 확대 / 축소       |

### 🔒 시크릿 모드

| 키     | 동작               |
| ----- | ---------------- |
| ← / → | 시크릿 이미지 이동       |
| R     | 복구 (원본 이미지로 되돌림) |
| ESC   | 일반 모드 복귀         |
| = / - | 확대 / 축소          |

---

## 7. 암호화 구조 (현재)

⚠️ 현재는 **단순 XOR 기반 임시 구조**

* 비밀번호 숫자 문자열
* 각 문자 charCode 합산 → 0~255 키 생성
* 파일을 byte 단위 XOR
* 복호화 시 동일 키 사용

※ 추후 AES-256 전환 예정
