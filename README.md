# My Secret Slide Viewer

**Free Image Viewer with Secret Storage**

macOS용 Electron 기반 **로컬 이미지 뷰어**  
→ 일반 이미지 감상 + **개인 전용 시크릿 이미지 관리(AES 암호화)** 를 위한 경량 앱

유료 이미지 뷰어의 과도한 기능·비용 없이  
**키보드 중심 / 오프라인 / 개인 사용**을 목표로 제작.

---

## 1. 프로젝트 개요

- 로컬 폴더 기반 이미지 뷰어
- 키보드 중심 조작
- 시크릿 이미지 **AES 암호화 저장**
- 네트워크 미사용 (완전 오프라인)
- 개인 단독 사용 전제

---

## 2. 기본 기능

### 📁 폴더

- 로컬 폴더 선택
- 지원 확장자
  - jpg
  - jpeg
  - png
  - gif
  - webp

---

### 🖼 이미지 뷰어

- ← / → : 이전 · 다음 이미지
- Space : 슬라이드쇼 재생 / 정지
- 자동 슬라이드 간격 : 2초
- 화면 중앙 정렬 / 전체 보기

---

## 3. 확대 / 이동 기능

### 🔍 확대 · 축소

| 키 | 동작 |
|---|---|
| `=` | 확대 |
| `-` | 축소 |

- 최소 배율 : 1
- 최대 배율 : 5

### 🧭 이동 규칙

- **확대 상태 (scale > 1)**
  - 방향키 → 이미지 내부 이동
- **확대되지 않은 상태 (scale = 1)**
  - ← / → → 이미지 전환

> 배율이 1로 돌아오면 위치 자동 초기화

---

## 4. 시크릿 모드 개념

### 🔐 시크릿 파일

- 확장자 : `.secret`
- 원본 이미지를 **AES로 암호화한 로컬 파일**
- 일반 모드에서는 **완전히 숨김**
- 시크릿 모드에서만 접근 가능

### 📂 모드 구분

| 모드 | 표시 대상 |
|---|---|
| 일반 모드 | 일반 이미지 파일 |
| 시크릿 모드 | `.secret` 파일 |

---

## 5. 키패드 & 비밀번호 흐름

### 키패드 호출 목적

| 목적 | 설명 |
|---|---|
| `ENCRYPT` | 현재 이미지를 시크릿 파일로 암호화 |
| `ENTER_SECRET` | 시크릿 모드 진입 |

### 비밀번호 입력 후 동작

| 호출 목적 | 결과 |
|---|---|
| ENCRYPT | 현재 이미지를 `.secret` 파일로 변환 |
| ENTER_SECRET | 시크릿 모드 진입 |

### 설계 원칙

- `H` 키 : **암호화 전용**
- `S` 키 : **시크릿 모드 진입 전용**
- 암호화 시 **모드 전환 없음**
- 비밀번호 실패 시 **아무 반응 없음**

---

## 6. 단축키 정리

### 📌 일반 모드

| 키 | 동작 |
|---|---|
| ← / → | 이미지 이동 |
| Space | 슬라이드쇼 재생 / 정지 |
| H | 현재 이미지 암호화 |
| S | 시크릿 모드 진입 |
| = / - | 확대 / 축소 |
| Delete / Backspace | 현재 이미지 삭제 (확인 후) |

---

### 🔒 시크릿 모드

| 키 | 동작 |
|---|---|
| ← / → | 시크릿 이미지 이동 |
| R | 복구 (원본 이미지로 복호화) |
| ESC | 일반 모드 복귀 |
| = / - | 확대 / 축소 |
| Delete / Backspace | 시크릿 이미지 삭제 |

---

## 7. 암호화 구조 (현재 구현)

### 🔑 AES 기반 암호화

- 비밀번호 → **키 파생 함수**를 통해 AES 키 생성
- 이미지 파일을 **AES 알고리즘으로 암호화**
- 암호화 시:
  - 원본 파일 삭제
  - `.secret` 파일 생성
- 복구 시:
  - AES 복호화
  - 원본 이미지 파일 복원

### 특징

- 로컬 파일 시스템 내에서만 동작
- 네트워크 전송 없음
- 개인 사용 목적의 보안 구조
